I plan to use a modern and simple color scheme, with mostly black and white elements and a few accent colors such as dark turquoise and Tomato.I will add on or change colors later if necessary The chosen font for the website is Monserrat sans-serif, which is a modern and legible font commonly used in web design. The Navigation Bar uses black text on a transparent background, while the Heading 2 also uses Monserrat sans-serif in black. The background of the website is off-white, with a dark turquoise border that has a font size of 200px. Lastly, social media links on the website are in Tomato-colored text on a transparent background. Overall, the typography and colors used in the website give a modern and clean look.
